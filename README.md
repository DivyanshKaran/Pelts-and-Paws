🔗 **Live:** [peltsandpaws.divyanshkaran.dev](http://peltsandpaws.divyanshkaran.dev)


**Pelts & Paws** is a full-stack pet management platform where pet owners can register, add pets, track health records, and earn gem rewards. The backend is built with **Java 21** and **Spring Boot 3**, featuring JWT authentication, role-based access control, and a clean layered architecture with five core JPA entities. The React + TypeScript frontend connects to a fully custom REST API with global exception handling and consistent error responses.
